# my_vim_config
### Install these packages on computer first
    - clangd
    - nodejs
    - npm yarn (in coc plugin dir)


### VIM Plugins
    - Plug (plugin handler)
    - Gruvbox
    - Autopair
    - Nerdtree
  
In VIM run `:PlugInstall`  
This will install all plugins in VIM.  
`Ctrl-b` will toggle Nerdtree.
